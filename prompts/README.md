# Site-Specific Content Prompts

This directory contains detailed content creation prompts for each site. When asked to write articles or content for a specific site, reference the appropriate prompt file.

## Available Prompts

| File | Site | Description |
|------|------|-------------|
| `dr-amy-heart-articles.md` | dr-amy | Article writing guidelines for Dr. Amy Heart's menopause/wellness site |

## Usage

When a user asks to write content for a site:
1. Read the corresponding prompt file
2. Follow all guidelines (voice, structure, widget rules, etc.)
3. Ask for the specific content title/topic
4. Create content following the prompt specifications

## Adding New Site Prompts

Create a new file following the naming convention:
```
{site-subdomain}-{content-type}.md
```

Examples:
- `dr-amy-heart-articles.md` - Articles for dr-amy site
- `dr-amy-heart-emails.md` - Email sequences for dr-amy site
- `wellness-hub-articles.md` - Articles for a different site

## Prompt Structure

Each prompt should include:
1. **Persona** - Who is the author/voice?
2. **Voice** - Tone and style guidelines
3. **Structure** - Content structure/outline
4. **Rules** - Hard constraints
5. **Widget Guidelines** - Which widgets to use and when
6. **Pre-writing Checklist** - What to identify before writing
